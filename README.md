# Number-Guessing-Game
A simple, fun, and interactive **Number Guessing Game** built using **HTML, CSS, and JavaScript**.  
Try to guess the correct number between **1 and 100** in limited attempts!

## ✨ Features

- 🎨 Modern UI with gradient + glassmorphism design  
- 🎯 Random number generation (1–100)  
- 📊 Attempts counter (max 10 tries)  
- 🏆 Score system  
- ⚡ Instant feedback (Too High / Too Low / Correct)  
- 🔄 Restart game option  
- 📱 Fully responsive design  
- 💀 Game Over feature
